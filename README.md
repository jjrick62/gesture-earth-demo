# Gesture Earth Demo

基于 MediaPipe Hands + Three.js 的手势控制 3D 地球 Demo。

## 启动方式

```bash
npx http-server . -p 8082
```

浏览器打开 `http://localhost:8082`，允许摄像头权限。

> 需要 HTTPS 或 localhost 才能使用摄像头 API。

## 手势操作

| 手势 | 动作 |
|---|---|
| 🖐 张开手掌移动 | 旋转地球 |
| 🤏 拇指食指捏合 | 缩放地球 |
| ✊ 握拳（持续半秒） | 暂停/恢复自转 |
| ☝ 单食指指向 | 在地球表面标记位置 |

## 键盘快捷键

| 键 | 功能 |
|---|---|
| `+` / `-` | 调节灵敏度 |
| `R` | 重置灵敏度 |
| 空格 | 暂停/恢复地球自转 |

## 降级方案

如果摄像头不可用，自动切换到键鼠模式：拖拽旋转、滚轮缩放、空格切换自转。

## 技术栈

- Three.js v0.160 — 3D 粒子地球
- MediaPipe Hands v0.4 — 手部关键点识别
- 手势分类 — 纯几何规则（指尖距离/角度）
